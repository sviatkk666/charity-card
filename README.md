# Charity Card

Односторінковий сайт-візитка для благодійного вечора — з унікальним анімованим дизайном.

> Проєкт на ранній стадії: назва події та фінальний дизайн ще не визначені.
> Дизайн планується створювати за допомогою **Claude Design**.

## Стек

Поки що базовий статичний односторінковий сайт (HTML / CSS / JS). Стек може змінитися після етапу дизайну.

## Запуск локально

Відкрий `index.html` у браузері, або запусти простий локальний сервер:

```bash
python3 -m http.server 8000
# відкрий http://localhost:8000
```

## Структура

```
.
├── index.html      # головна (і єдина) сторінка
├── styles/         # стилі
├── scripts/        # JS / анімації
└── assets/         # зображення, шрифти, медіа
```
